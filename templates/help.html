• /start - launch a bot
  • /getchannels - get channels list with name and id
  • /addchannel - add channels which will get answers from user
  • /deletechannel - disable channel for getting answers from user
  • /getuser - get channels list with name and id
  • /adduser - add user(s) to ignore list
  • /deleteuser - remove user(s) from ignore list
